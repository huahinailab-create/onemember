Points campaigns reward spending — for example "1 point per 10 baht".

1. Open **Campaigns** and press **Create campaign**.
2. Choose **Points**.
3. Set how many points customers earn per amount spent.
4. Name the campaign and press **Save**, then set it **Active**.

From now on, every purchase you record adds points automatically. Add rewards next so members have something to aim for.
