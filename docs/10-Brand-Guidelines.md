# 10 — Brand Guidelines

This document is the authoritative brand reference for OneMember.

All product copy, UI text, marketing materials, and documentation must conform to these guidelines.

---

## Brand Name

**OneMember**

Always written as one word with a capital O and capital M.

| Correct | Incorrect |
|---|---|
| OneMember | ONE MEMBER |
| | One Member |
| | onemember |
| | Onemember |
| | one member |

**Reward loyalty. Grow your business.**

Usage notes:
- Use the full tagline on landing pages, email headers, and promotional materials.
- The full stop after each sentence is intentional.
- Never truncate to a single sentence.

---

## Tone of Voice

All product copy, help text, error messages, success messages, and communications must follow these principles:

- **Professional** — correct grammar, no slang, no emojis in functional UI.
- **Friendly** — use plain language; speak directly to the merchant as "you".
- **Clear** — one idea per sentence; avoid jargon.
- **Encouraging** — celebrate merchant progress; never blame the user for errors.
- **Never overly technical** — avoid database terms, API references, or implementation details in user-facing copy.

### Examples

| Avoid | Prefer |
|---|---|
| "The record was persisted successfully." | "Member saved successfully." |
| "An error occurred during validation." | "Please check the highlighted fields and try again." |
| "Your account has been soft-deleted." | "Member archived. You can view archived members any time." |
| "Enter the member's phone number (unique per merchant_id)." | "Enter the member's mobile number." |
