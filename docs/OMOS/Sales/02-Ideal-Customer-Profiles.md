# 02 — Ideal Customer Profiles (ICPs)

> Ten segments, one page each in table form. **Fit first, then pitch.** If you see two or more red flags, qualify out politely and keep the relationship.

**Universal ICP baseline (all segments):** walk-in repeat business matters · owner reachable · smartphone in daily use · 20+ customers/day or high-ticket repeat cycle · shop expects to exist in 12 months.

**Universal red flags:** pure tourist trade (no repeats possible) · owner permanently absent AND manager disempowered · cash-only *and* smartphone-averse ownership · shop visibly closing/for-rent · franchisee with no authority over marketing (route to franchisor as Enterprise lead instead).

---

## ☕ Coffee Shop — *primary beachhead*
| | |
|---|---|
| Business size | 1–2 branches, 2–6 staff, 80–300 cups/day, ticket ฿60–150 |
| Typical owner | 28–45, opened from passion, lives on IG/LINE, does everything herself (P1) |
| Pain points | Can't tell regulars from passers-by; paper stamps lost/forged; weekday afternoons dead |
| Buying triggers | Competitor loyalty next door; new branch; rainy-season dip |
| Ideal modules | **Stamps** campaign, QR poster, Counter Mode; birthday reward later |
| Red flags | Office-tower kiosk with 100% captive audience (no churn to fight); owner about to sell shop |

## 🍜 Restaurant
| | |
|---|---|
| Business size | Family-run to 20 staff; ticket ฿150–600; delivery-app dependent |
| Typical owner | 35–60; P1 or absent-owner + P2 manager; margins squeezed by platforms |
| Pain points | 30% delivery commissions; no direct channel to customers; empty Mon–Thu |
| Buying triggers | Delivery fee increase; want direct ordering; new competitor |
| Ideal modules | **Points**, win-back alerts, **Commerce storefront** (direct orders), QR table tents |
| Red flags | Pure delivery cloud kitchen (no counter, no walk-ins); staff forbidden phones at service |

## 💇 Hair Salon
| | |
|---|---|
| Business size | 1 shop, 2–8 chairs, ticket ฿150–800, 4–8 week visit cycle |
| Typical owner | Owner-stylist (P1) — her hands are busy all day; admin time ≈ zero |
| Pain points | Clients follow departing stylists; no-shows; can't remember who's overdue |
| Buying triggers | Stylist just quit; slow month; competitor promos |
| Ideal modules | **Points** + **birthday**, member notes, **win-back at 45–60 days** |
| Red flags | Chair-rental model (stylists are the business, shop is landlord — no shared customer list) |

| | |

## 💆 Massage & Spa
| | |
|---|---|
| Business size | 4–20 beds, ticket ฿250–1,500, tourist/local mix |
| Typical owner | Often older ownership + front-desk manager (P2); cash culture |
| Pain points | Tourists vs locals indistinguishable; therapists paid per session (tracking pain); local regulars under-nurtured |
| Buying triggers | High-season prep; corporate/hotel package deals; new branch |
| Ideal modules | **Points**, rewards ladder (30/60/100 sessions), Counter Mode per front desk |
| Red flags | >80% tourist trade; owner rejects any customer data collection |

## 🏨 Hotel (boutique/independent)
| | |
|---|---|
| Business size | 10–60 rooms, F&B attached, OTA-driven |
| Typical owner | P3/P4 — ops manager runs evaluation; owner signs |
| Pain points | OTA commission + zero guest ownership; no repeat mechanism; F&B underused by guests |
| Buying triggers | Direct-booking initiative; repeat-guest program mandate |
| Ideal modules | Points on stays/F&B, vouchers via storefront, **Enterprise** (roles, reporting) |
| Red flags | Expects PMS integration as condition #1 (we don't have it — do not promise); full-service chains (procurement we can't win yet). **Longest cycle — never a week-1 target** |

## 🛍️ Retail (minimart, gift, pharmacy-adjacent)
| | |
|---|---|
| Business size | 1–3 shops, thin margins, ticket ฿50–300 |
| Typical owner | Pragmatic, numbers-driven, 7-Eleven-scarred (P1/P3) |
| Pain points | Basket stagnation; zero differentiation vs chains |
| Buying triggers | Rent renewal panic; chain store opening nearby |
| Ideal modules | **Points per amount**, expiry-pressure campaigns, member-only prices via rewards |
| Red flags | Sub-฿50 average ticket AND <50 customers/day (points feel meaningless); wet-market stall dynamics |

## 💉 Beauty Clinic
| | |
|---|---|
| Business size | 2–15 staff, ticket ฿1,000–20,000, course/package-based |
| Typical owner | Doctor-owner or investor + clinic manager (P2/P4); compliance-aware |
| Pain points | Course completion drop-off; referral dependence; CRM too heavy or absent |
| Buying triggers | New machine ROI pressure; formalizing referrals; second branch |
| Ideal modules | Points on treatments, reward = free add-on service, **Enterprise** for chains |
| Red flags | Wants medical-record integration (we are not a medical system — say so clearly); procurement longer than 3 months |

## 👗 Fashion (boutique + IG hybrid)
| | |
|---|---|
| Business size | 1 shop + IG/LINE shop, drop-based sales spikes |
| Typical owner | Young, brand-conscious, follower-count-focused (P1) |
| Pain points | Followers ≠ repeat buyers; no view of who buys twice |
| Buying triggers | New collection launch; slowing IG organic reach |
| Ideal modules | **Points**, member-only early access (reward), storefront as catalogue |
| Red flags | Livestream-only sellers with no physical counter; resale/thrift one-off inventory (repeats structurally rare) |
