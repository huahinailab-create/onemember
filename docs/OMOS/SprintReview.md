## Sprint Review Template

When a sprint is complete, the ChatGPT CTO replaces the content above with the following structure:

```markdown
### Sprint [ID] — [Title]

| Field | Value |
|---|---|
| **Sprint ID** | [ID] |
| **Review Date** | YYYY-MM-DD |
| **Reviewer** | ChatGPT CTO |
| **Rating** | ✅ Approved / ⚠️ Approved with notes / 🔄 Revision required / ❌ Rejected |

#### Objective Alignment
[Did the sprint achieve what it said it would? Evidence?]

#### Code Quality
[Findings on code quality, patterns, naming, etc.]

#### Security
[Security checklist review]

#### Testing
[Test count, coverage assessment, quality of regression tests]

#### Documentation
[Quality and completeness of any documentation produced]

#### Recommendations
[Notes for the next sprint or outstanding items]

#### Decision
[Rating with rationale. If not Approved: what must change before approval?]
```
