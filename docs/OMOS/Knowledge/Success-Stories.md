---

## Purpose

Documented merchant success stories are evidence that OneMember works. They are used to:
- Validate product decisions (does this feature create real value?)
- Inform marketing and sales conversations
- Guide new feature development (what do successful merchants have in common?)

Format:
```
### SS-[NNN] — [Merchant Name or Type]
**Date:** YYYY-MM-DD
**Merchant Type:** [Cafe / Restaurant / Salon / etc.]
**Metric Before:** [What their situation was before OneMember]
**Metric After:** [What changed after OneMember]
**Key Feature Used:** [What OneMember feature drove the result]
**Quote:** [Direct merchant quote if available]
```
