## Purpose

---

## Interview Template

Each interview record should follow this structure:

```markdown
### Interview [NNN]

| Field | Value |
|---|---|
| **Date** | YYYY-MM-DD |
| **Merchant Type** | [Cafe / Restaurant / Salon / Retail / Hotel / Enterprise] |
| **Location** | [City / Area] |
| **Business Size** | [Number of locations, estimated monthly customers] |
| **Current Loyalty** | [Paper / LINE / App / Nothing] |
| **Interviewed By** | [Name / Role] |

#### Key Insights
- [Insight 1]
- [Insight 2]

#### Pain Points Mentioned
- [Pain point 1]
- [Pain point 2]

#### Features They Wanted
- [Feature 1]
- [Feature 2]

#### Willingness to Pay
[What they said about budget and pricing]

#### Follow-Up
[Any agreed next steps or follow-up actions]
```
