## Purpose

Unprocessed feature ideas and product observations. These are not commitments — they are inputs to the Decision Framework.

---

## How to Use This File

Add ideas here when they occur — from merchant conversations, customer observations, competitor analysis, or team discussion. An idea in this file is NOT a commitment. It is an input.

Before any idea moves to the backlog, it must be evaluated using the Decision Framework scoring model (`00-Executive/Decision-Framework.md`).

Format each idea as:
```
### IDEA-[NNN] — [Title]
**Source:** [Who suggested it / where it came from]
**Date:** YYYY-MM-DD
**Description:** [What the idea is]
**Initial Assessment:** [First instinct — worth exploring? Which phase?]
```
